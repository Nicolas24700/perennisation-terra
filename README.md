## ⚙️ Installation

1. Recuperer le projet :
	- soit en telechargeant le fichier ZIP puis en l'extrayant
	- soit avec Git :

```bash
git clone https://github.com/Nicolas24700/perenisation-terra.git
cd perenisation-terra
```

2. Verifier que les dossiers `fonts` et `img` sont bien presents a la racine du projet.

## 🌐 Lancer le site en local

Ouvrir directement le fichier `index.html` dans votre navigateur.
